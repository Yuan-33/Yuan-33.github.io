MTP

Multi-Token Prediction — 训推一体加速
Multi-Token Prediction（MTP）是一种训练范式，让模型在每个位置同时预测未来多个 token 而非仅预测下一个。它既能作为训练目标提升模型质量，又能在推理时将 MTP 头复用为 Draft 模型实现投机解码加速，DeepSeek-V3 率先将这一理念工程化落地。

关键特性

  • 多位置预测：在每个 token 位置同时预测 t+1, t+2, ..., t+k 的输出，提供更密集的梯度信号
  • 训练质量提升：MTP 作为辅助训练目标，强制模型学习更丰富的内部表征，尤其对大模型效果显著
  • 训推一体：训练阶段的 MTP 模块在推理时直接复用为 Draft Head，无需额外部署 Draft 模型
  • DeepSeek-V3 落地：DeepSeek-V3 使用 MTP 实现约 1.8x TPS 提升，验证了大规模工程可行性
  • 参数高效：MTP 头共享主模型的 Embedding 层和部分 Transformer 层，额外参数开销小
  • 自回归 Draft：MTP 头以自回归方式生成 Draft 序列，保持 token 间的依赖关系，接受率更高
