Eagle-v1

Speculative Decoding — Feature-Level Draft
Eagle（Extrapolation Algorithm for Greater Language-model Efficiency）是一种基于特征级别的投机解码方法，由 SafeAI Lab 提出。它使用轻量级 Draft Head 从主模型的隐藏状态（而非 token embedding）预测未来 token，实现了比传统 token-level 方法更高的接受率，端到端加速 2.5-3.5 倍。

核心原理

传统投机解码（如 Medusa）在 token 级别进行 Draft，而 Eagle 发现 token embedding 空间的不确定性远高于特征（hidden state）空间。因此 Eagle 在主模型最后一层的隐藏状态上训练一个轻量级 Draft Head（线性层 + Transformer Block），以自回归方式在特征空间中生成 Draft 序列，显著提升了预测准确性。

关键特性

  • 轻量级 Draft Head：仅一个线性投影层 + 一个 Transformer Block，参数量不到主模型的 1%，训练和推理开销极小
  • 特征空间 Draft：在隐藏状态空间而非 token 空间进行预测，利用了特征空间不确定性更低的特性，接受率比 token-level 方法高 20-40%
  • 高接受率：在 MT-Bench、HumanEval 等基准上，Draft 接受率显著高于 n-gram、Medusa 等方案，尤其在复杂推理任务上优势明显
  • 无损加速：通过拒绝采样保证输出分布与主模型完全一致，加速不牺牲任何质量
  • 部署简单：无需独立部署 Draft 模型，Draft Head 与主模型共享前向传播流程，单卡即可运行
