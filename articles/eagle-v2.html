Eagle-v2

Speculative Decoding — Dynamic Draft Tree
引入动态 draft tree 结构，根据 token 概率自适应调整树的宽度和深度，显著提升投机接受率。

关键特性

  • 动态树构建：根据 draft token 概率动态调整树的形状，高概率路径更深
  • Tree Attention：一次前向验证整个 draft tree，最大化 GPU 利用率
  • 接受率相比 Eagle-v1 提升 20-40%，端到端加速比更高
  • 可配置树大小，在延迟与吞吐间灵活权衡
