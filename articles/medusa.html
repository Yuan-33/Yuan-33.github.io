Medusa

Speculative Decoding — Multiple Decoding Heads
Medusa 是一种无需独立 Draft 模型的投机解码方法，通过在冻结的 LLM 最后一层隐藏状态上添加多个轻量级预测头（Medusa Heads），并行预测未来多个位置的 token，配合 Tree Attention 验证机制实现 2-3 倍推理加速。

关键特性

  • 多头并行预测：在主模型最后一层隐藏状态上叠加多个 Medusa Head，每个头独立预测不同位置的 token
  • 冻结主干模型：Medusa-1 模式下仅训练新增的预测头，主模型参数完全不动，训练成本极低
  • Tree Attention 验证：将多个候选序列组织为树状结构，通过一次前向传播验证所有分支，最大化 GPU 利用率
  • 即插即用：兼容 LLaMA、Mistral、Vicuna 等主流架构，无需修改原始模型结构
  • 2-3x 加速：在 Vicuna-7B/13B/33B 上实现 2.2x-3.0x 端到端加速，质量损失可忽略
  • Medusa-2 联合训练：进一步微调主干模型与预测头，接受率更高但需要更多训练资源
