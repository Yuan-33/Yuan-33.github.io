│   ├── 三级标题（直接链接到页面）
│   └── ...

└── ...

## 🔧 开发规范

### HTML 页面模板

每个 `.html` 文件应包含：
1. **统一的导航栏**
2. **左侧导航树**（点击展开）
3. **主内容区**（使用语义化 HTML）
4. **响应式布局**（Flexbox/Grid）
5. **深色/浅色主题支持**

### CSS 规范

- 使用 CSS 变量管理主题颜色
- 支持 `data-theme="dark"` 和 `data-theme="light"`
- 所有颜色通过变量定义（便于主题切换）
- 使用 Flexbox 实现响应式布局

### JavaScript 规范

- 所有交互逻辑集中管理
- 使用事件委托处理导航
- 确保移动端和桌面端都正常工作
- 性能优先（避免大量 DOM 操作）
